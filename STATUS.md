# Status

*Stunspot's Guide to Investigative News Intelligence* is at **release 1.0**.

This is a complete public release package for a model-facing investigative news intelligence canon. It is intended for AI/RAG ingestion, long-context use, project knowledge, research support, and human reference.

## Release Facts

- Version: **1.0**
- Release date: **2026-06-28**
- Source reports: **10**
- Compiled packs: **4**
- Omnibus files: **1**
- License: **CC BY-NC-SA 4.0**

## Corpus State

The individual source-report corpus lives in `knowledge-packs/by-report/` and currently contains reports **A–J**.

The compiled and omnibus packs represent the broader **A–M** canon sequence. Vol. 4 contains **K–M: Editorial Control, Failure Diagnosis, and Execution**.

`docs/` is the navigation and guidance layer only. It is not the report corpus and this repository does not use `docs/reports/`.

## Maturity

This release is suitable for loading into AI projects, retrieval systems, long-context sessions, and research workflows as a stable reference corpus.

It should still be treated as a knowledge artifact rather than a live database. For current-event analysis, pair the canon with fresh primary sources, public records, reporting, platform data, and explicit citation.

## Verification Note

The corpus is designed to support investigative reasoning, evidence hygiene, and model instruction. It does not replace independent verification of high-impact factual claims.
