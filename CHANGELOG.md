# Changelog

## v1.0 — 2026-06-28

Initial public release of *Stunspot's Guide to Investigative News Intelligence*.

- Added 10 individual source reports under `knowledge-packs/by-report/`.
- Added 4 compiled upload packs under `knowledge-packs/compiled-packs/`.
- Added 1 omnibus file under `knowledge-packs/omnibus/`.
- Added GitHub Pages navigation and usage guidance under `docs/`.
- Added `MANIFEST.md` and `manifest.json` with source-to-output mappings, corpus counts, file paths, and release metadata.
- Added `CITATION.cff`, `STATUS.md`, and `LICENSE.md` for citation, release-state, and license clarity.

Notes:

- `docs/` is navigation and guidance only; the report corpus lives in `knowledge-packs/`.
- The individual source-report directory contains A–J. The compiled and omnibus packs include the broader A–M canon sequence, including Vol. 4 K–M.
- Reserved hero/social image references are retained for later brand assets.
